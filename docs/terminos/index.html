Términos y condiciones

Al usar WeatherVar aceptás que la información meteorológica se ofrece con fines informativos y orientativos.

No reemplaza canales oficiales

WeatherVar no debe utilizarse como única fuente para decisiones de seguridad, navegación, aviación, operaciones críticas o emergencias.

Disponibilidad

El sitio puede cambiar, interrumpirse o mostrar datos incompletos si alguna fuente pública no está disponible.

Uso permitido

Podés consultar y compartir enlaces del sitio. No está permitido usar la marca WeatherVar para simular comunicación oficial del SMN o de autoridades públicas.

Volver al inicio

WeatherVar es un sitio independiente. Información meteorológica orientativa basada en datos públicos y fuentes oficiales, incluyendo el Servicio Meteorológico Nacional. Ante fenómenos severos, consultá siempre los canales oficiales.
